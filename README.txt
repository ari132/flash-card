To run the app:
1) Open a shell in this folder
2) Create a virtual environment: python3 -m venv .env
3) Activate the virtual environment: source .env/bin/activate
4) install the requirements from requirements.txt: pip install -r requirements.txt
5) Run the app.py file: python app.py

Use the following values for logging in:
1) username:root	password:root
2) username:test	password:test
